# CanBeWell Landing Page

CanBeWell 移动健康平台的官方网站。

## 关于 CanBeWell

CanBeWell 是一个灵活的移动健康平台，它将干预内容与应用逻辑解耦，使临床医生能够设计、审查和部署个性化干预措施，而无需重新编译应用。

CanBeWell 集成了多模态数据收集——可穿戴设备、生态瞬时评估（EMA）和智能手机传感器——与无代码干预创作界面相结合，为医疗健康领域提供创新的解决方案。

## 功能特点

- **解耦架构**: 干预内容与应用逻辑完全分离，实现灵活的内容管理
- **无代码创作**: 临床医生无需编程知识即可设计个性化干预方案
- **多模态数据**: 集成可穿戴设备、EMA评估和智能手机传感器数据
- **实时部署**: 无需重新编译应用即可部署新的干预内容
- **智能收集**: 自动收集被动和主动数据，全面了解患者健康状况
- **个性化干预**: 基于患者数据提供精准的个性化健康干预

## 系统工作流程

1. **数据收集模块**: 从 Fitbit 和 EMA 调查收集被动和主动数据
2. **临床医生仪表板**: 数据存储、可视化和干预内容创建
3. **干预交付模块**: 将个性化干预内容推送给患者

## 部署

这个 landing page 可以通过 GitHub Pages 部署：

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为源

或者使用任何静态网站托管服务。
